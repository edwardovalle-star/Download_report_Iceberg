# Guía rápida para usuario final

## 1. Abrir la aplicación

Ingresar a la URL de Streamlit Cloud entregada para la app.

## 2. Iniciar sesión

Ingresar usuario y contraseña de ICEBERG.

## 3. Seleccionar periodos

Escoger los periodos a consultar.

## 4. Ejecutar proceso

Presionar el botón de descarga/consolidación.

## 5. Aplicar filtros

Seleccionar dependencia, fecha y condiciones de filtrado.

## 6. Descargar resultados

La app permite descargar:

- Archivo seleccionado.
- ZIP completo de la ejecución.

## Recomendaciones

- No compartir credenciales.
- No publicar resultados si contienen datos sensibles.
- Revisar periodo y filtros antes de descargar.
